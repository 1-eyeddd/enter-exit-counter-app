# README

### version
- Rails 6系
- node.js v18系

### setup

```
$ git clone https://github.com/1-eyeddd/enter-exit-counter-app.git
$ cd enter-exit-counter-app
$ rails db:migrate
$ rails webpacker:install
```

### 編集場所

```
/app/views/counters/index.html.erb(混雑表示画面)
/app/controllers/counters_controller.rb(コントローラー)
```
